## Accessibility Profiles (Presets)

### Motor Impaired
Optimized interaction using keyboard navigation and enlarged clickable areas.

### Blind Users (Screen Reader)
Automatic adjustments for compatibility with:
- JAWS
- NVDA
- VoiceOver
- TalkBack

> NOTE: This profile auto-prompts for screen readers.

---

### Dyslexia Profile
- Dyslexia-friendly fonts
- Increased spacing
- Reduced visual noise

---

### Cognitive & Learning Disabilities

#### ADHD Friendly Profile
- Reduces distractions
- Emphasizes primary content
- Improves focus and reading flow

#### Cognitive Disability Profile
Supports users with:
- Autism
- Dyslexia
- CVA
- Learning disabilities  
by simplifying layout and reducing cognitive load.

---

### Vision Impaired Profile
Optimizes visibility for:
- Cataracts
- Glaucoma
- Tunnel vision
- Degrading eyesight

---

### Seizure Safe Profile
Prevents seizure risks by:
- Eliminating flashing/blinking animations
- Neutralizing risky color combinations

---

## Keyboard Navigation (Motor Accessibility)

- Full keyboard operation:
  - Tab / Shift + Tab
  - Enter
- Shortcuts:
  - `M` → Menus
  - `H` → Headings
  - `F` → Forms
  - `B` → Buttons
  - `G` → Graphics

> NOTE: Auto-prompted for keyboard users.

## Advanced AI & Compliance Additions

### AI Content Tools
- **AI Auto-Alt Text**  
  Computer vision based automatic image descriptions.

- **Text Simplifier (Cognitive Ease)**  
  Converts complex language into plain, easy-to-read text.

- **Virtual Sign Language Avatar**  
  Digital avatar translating content into sign language.

---

### Vision & Focus Enhancements
- **Reading Mask (Focus Ruler)**  
  Horizontal reading aid for line-by-line reading.

- **Advanced Color Blindness Filters**
  - Protanopia
  - Deuteranopia
  - Tritanopia

- **Magnifier Glass**
  Cursor-based magnification tool.

---

### Interaction & Audio
- **Voice Navigation**
  Voice commands like:
  - “Scroll down”
  - “Open menu”

- **Audio Feedback (Sound Cues)**
  Sounds on:
  - Button interaction
  - Link focus
  - Form submission

- **Focus Highlighter Customizer**
  Custom color & thickness for focus rings  
  (WCAG 2.2 compliant)
